## 良いコンセプトの定義（2つ）

### 定義1: リフレーミング公式

```
（既存の概念）を（新しい視点）で捉え直し、
（顧客のインサイト）に応える、（短く伝わる言葉）
```

**4要素:**
| 要素 | 説明 |
|-----|------|
| リフレーミング | 既存の常識を新しい視点で捉え直す |
| 顧客インサイト | 潜在的な欲求や葛藤を解決 |
| シンプルで直感的 | 短く伝わりやすい |
| 行動を生み出す | 試したくなる言葉 |

---

### 定義2: いい商品コンセプトの5条件

*出典: HADO マーケティングブートキャンプ*

| # | 条件 | ポイント |
|---|------|---------|
| 1 | お金を払ってでも手に入れたいもの | 需要とコストのバランス |
| 2 | 新しさや意外性が入っている | 需要の半歩先を行く |
| 3 | ベネフィットが入っている | 顧客が得られる価値 |
| 4 | ベネフィットを信じてもらえる | 理解できるロジック |
| 5 | シンプルである | 分かりやすさ |

---

## 定義の使い分け

| 観点 | 定義1（リフレーミング公式） | 定義2（5条件） |
|-----|--------------------------|---------------|
| 焦点 | ネーミング・タグライン | 商品コンセプト全体 |
| 強み | 差別化・独自性 | 購買までの説得力 |
| 使いどころ | コピー・ネーミング開発 | 商品企画・広告設計 |
