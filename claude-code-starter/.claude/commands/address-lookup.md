# /address-lookup - 住所検索・フォーム変換

企業のWebサイトから住所を取得し、海外通販サイトの配送フォームに入力可能な形式（英語）に変換します。

## 前提条件

- 企業のWebサイトURL（または企業名）
- 対象国（デフォルト: Japan）

## 実行手順

### 1. 企業サイトから住所を取得

1. 提供されたURLのトップページを `WebFetch` で取得
2. 住所情報がない場合、以下のページを探索:
   - `/offices/`
   - `/company/`
   - `/about/`
   - `/contact/`
   - `/access/`
3. 完全な住所情報を抽出:
   - 郵便番号
   - 都道府県
   - 市区町村
   - 番地
   - ビル名・階数
   - 電話番号（あれば）

### 2. 配送フォーム形式に変換

日本の住所を以下の形式に整形:

| Field | 内容 |
|-------|------|
| Postal code | 郵便番号（ハイフン付き） |
| Prefecture | 都道府県（英語） |
| City | 市区町村（ローマ字） |
| Address | 番地（ローマ字、逆順） |
| Apartment/Suite | ビル名・階数（英語） |
| Phone | 電話番号（国際形式） |
| Company | 会社名（英語） |

### 3. 結果を出力

**日本語版**と**英語版**の両方をテーブル形式で出力。

## 変換ルール

### 都道府県の英語表記

| 日本語 | 英語 |
|--------|------|
| 東京都 | Tokyo |
| 大阪府 | Osaka |
| 北海道 | Hokkaido |
| 京都府 | Kyoto |
| 〇〇県 | [名前] |

### 住所の英語表記

- 番地は数字のまま（例: 6-10-1）
- 地名はローマ字（例: Roppongi）
- 順序: 番地 → 地名（日本語と逆）
- 「丁目」「番地」「号」は省略

### ビル名の英語表記

| 日本語 | 英語 |
|--------|------|
| 〇〇ビル | [名前] Building |
| 〇〇タワー | [名前] Tower |
| 〇〇ヒルズ | [名前] Hills |
| 〇階 | [数字]F |

### 電話番号の国際形式

- 日本の場合: `+81-[市外局番（0除く）]-[番号]`
- 例: `03-1234-5678` → `+81-3-1234-5678`

## 出力例

```
## [企業名] 住所

### 日本語

| 項目 | 値 |
|------|-----|
| 郵便番号 | 106-6131 |
| 住所 | 東京都港区六本木6-10-1 六本木ヒルズ30階 |
| 電話 | 03-6384-5540 |

### English (Delivery Form)

| Field | Input |
|-------|-------|
| Postal code | 106-6131 |
| Prefecture | Tokyo |
| City | Minato-ku |
| Address | 6-10-1 Roppongi |
| Apartment, suite, etc. | Roppongi Hills 30F |
| Phone | +81-3-6384-5540 |
| Company | [Company Name] Inc. |
```

## 使用例

```
/address-lookup https://example.co.jp
/address-lookup [企業名]
```

## 注意事項

- 複数拠点がある場合は本社（または東京オフィス）を優先
- 住所が見つからない場合はその旨を報告
- 機密性の高い住所（個人宅等）は対象外
