# Q4計画の根拠 (Q4 Plan Reasoning)

q4.md の投資配分・BF連動・BestCosme戦略・年間達成検証の根拠・計算式を説明する。

---

## 1. Q4投資4.4億円（年間最大）の根拠

### 1.1 年間最大投資の理由

**四半期別投資比較**:

| 四半期 | 投資額 | 比率 | 特徴 |
|--------|--------|------|------|
| Q1 | 3.8億円 | 24% | 立ち上げ |
| Q2 | 3.6億円 | 23% | ジョングク（別枠あり） |
| Q3 | 4.0億円 | 25% | Drug Store参入 |
| **Q4** | **4.4億円** | **28%** | **年間最大、BF+BestCosme** |

**Q4を最大投資とする理由**:
```
1. 年間最大の商機:
   - 11月メガ割 = 年間最後のメガ割
   - Black Friday = 全ECチャネルが活性化
   - 年末ギフト需要 = 高単価セットが売れる

2. 認知の蓄積:
   - Q1-Q3で積み上げた認知を最大活用
   - ジョングク効果の継続（素材再利用）

3. BestCosme獲得:
   - @cosme施策に0.45億円投下
   - 来年への布石として重要
```

### 1.2 月別配分の根拠

| 月 | 予算 | 比率 | 役割 |
|----|------|------|------|
| 10月 | 1.0億円 | 23% | 準備・@cosmeレビュー獲得 |
| 11月 | 2.0億円 | 45% | メガ割 + BF全力 |
| 12月 | 1.4億円 | 32% | BestCosme + 年末ギフト |

**ロジック**:
```
11月（メガ割+BF）: 年間売上の最大ピーク
→ 投資集中（45%）

12月: BestCosme発表、年末需要
→ 11月ほどではないが維持（32%）

10月: 準備期間
→ 最小限だが@cosme対策は必要（23%）
```

## 2. 11月メガ割GMV 10億円の根拠

### 2.1 Q2（6月）との比較

| 要因 | Q2（6月） | Q4（11月） | 変化 |
|------|----------|-----------|------|
| ジョングク効果 | 最大（発表直後） | 継続（素材再利用） | → |
| 認知の蓄積 | 2回目メガ割 | 4回目メガ割 | ↑ |
| BF連動 | なし | あり | ↑ |
| 季節要因 | 初夏 | 年末前 | ↑ |
| Drug Store | なし | 稼働中 | ↑ |

**計算式**:

```
Q2 GMV: 10億円（ジョングク効果最大）
Q4 GMV: 10億円（複数要因で同等以上）

成長ドライバー:
+ 認知蓄積（4回目）: +10%
+ BF連動効果: +15%
+ 年末需要: +10%
- ジョングク効果減少: -20%
- Q3（8億円）からの反発: +5%

→ 10億円維持が妥当
```

### 2.2 「年間最後のメガ割」効果

**心理的効果**:
```
「今年最後のメガ割」訴求

効果:
1. 購買意欲の最大化: 「今買わないと来年まで待つ」
2. まとめ買い: 年末に向けてストック
3. ギフト需要: 「メガ割でお得に買ってプレゼント」
```

**競合データからの推測**:
- 11月メガ割は年間で最大売上を記録するブランドが多い
- BF連動が効く（Amazon/楽天からの流入）

## 3. Black Friday全チャネル連動の根拠

### 3.1 連動戦略の設計理由

**従来（チャネル単独）**:
```
Qoo10: メガ割のみ
Amazon: BFのみ
楽天: マラソンのみ
TTS: 通常運用

→ チャネル間でカニバリ、または機会損失
```

**提案（全チャネル連動）**:
```
メガ割終了（11/12頃）
     ↓
BF開始（11/21）
     ↓
全チャネルで「今年最後のセール」訴求
     ↓
Qoo10で買い逃した人 → Amazon/楽天/TTSで購入
```

### 3.2 BF期間の売上目標

**計算式**:

```
BF期間: 11/21〜12/1（11日間）

チャネル別BF売上:
- TTS: 1.0億円（日予算80-150万 × ROAS 200%）
- Amazon: 0.5億円（BF効果）
- 楽天: 0.2億円（ポイント効果）
- Qoo10追加: 0.3億円（メガ割終了後の追加需要）

BF期間合計: 2.0億円
```

### 3.3 タイムライン設計の根拠

| 日程 | イベント | 狙い |
|------|---------|------|
| 11/12頃 | Qoo10メガ割終了 | 刈り取り完了 |
| 11/13-20 | **中休み期間** | BF期待感醸成 |
| 11/21 | TTS BF Sale開始 | TTS先行でバズ |
| 11/24 | Amazon BF開始 | 最大商機 |
| 11/27 | Amazon BF終了 | - |
| 12/1 | 全BF終了 | 締め切り効果 |
| 12/2 | サイバーマンデー | 追い打ち |

**中休み期間の役割**:
```
メガ割終了後、すぐにBFがあることを認知
→ 「メガ割で買い逃しても大丈夫」
→ BFへの期待感が高まる
→ BF初日に購買爆発
```

## 5. 年末ギフト施策の根拠

### 5.1 ギフトセット設計の理由

**セット構成**:

| セット名 | 内容 | 価格 | ターゲット |
|---------|------|------|-----------|
| スターター | Serum（ミニ）+ サンプル | 2,000-3,000円 | 「試してみたい」層 |
| ベーシック | Serum + Toner | 5,000-6,000円 | 自分用/友人用 |
| プレミアム | Serum + Mask + Toner | 8,000-10,000円 | 彼女/母親用 |

**価格帯設定の根拠**:
```
美容ギフト市場の価格帯分布:
- 3,000円以下: カジュアルギフト（30%）
- 5,000円前後: 友人・同僚向け（40%）
- 10,000円前後: 家族・パートナー向け（30%）

→ 全価格帯をカバーする3セット構成
```

### 5.2 限定パッケージの効果

**施策**:
- クリスマス/年末限定デザイン
- ボックス入り（そのままプレゼント可能）
- リボン/メッセージカード付き

**期待効果**:
```
限定パッケージの効果:

1. SNS拡散:
   - 「かわいいパッケージ届いた」系投稿
   - 開封動画がバズりやすい

2. 単価UP:
   - 限定パッケージは通常より+500-1,000円高く設定可能
   - ギフト需要で価格感度が下がる

3. リピート獲得:
   - プレゼントされた人が自分用に購入
   - 新規顧客獲得チャネル
```

## 7. 年間73.6億円達成の検証

### 7.1 四半期別GMV積み上げ

| 四半期 | 目標GMV | 達成確度 | 備考 |
|--------|---------|---------|------|
| Q1 | 5-7億円 | 中 | 新規参入リスク |
| Q2 | 10億円 | 高 | ジョングク効果 |
| Q3 | 8億円 | 中 | Drug Store参入 |
| Q4 | 10億円 | 高 | 年間最大商機 |
| **合計** | **33-35億円** | | メガ割のみ |

**年間73.6億円の内訳**:

```
メガ割売上: 33億円（保守的）
通常期EC売上: 月0.5-1.0億円 × 8ヶ月 = 4-8億円
TikTok Shop売上: 10億円（年間）
Offline売上: 25-30億円（Drug Store + Variety Store）

合計: 72-81億円
中央値: 73.6億円
```

### 7.2 リスクシナリオ

| シナリオ | GMV | 条件 |
|---------|-----|------|
| 楽観 | 85億円 | 全施策成功、ジョングク効果継続 |
| **基本** | **73.6億円** | 計画通り |
| 保守 | 60億円 | Q1不振、Drug Store遅延 |
| 悲観 | 45億円 | 複数リスク発生 |

**リスク軽減策**:
```
Q1不振の場合:
→ Q2のジョングク効果で挽回
→ 年間目標は維持可能

Drug Store遅延の場合:
→ Offlineの25-30億円が減少
→ EC強化で補填（TTS日予算増額等）
```

### 7.3 年間ROAS 466%の達成確認

**計算式**:

```
投資額: 15.8億円（$10M枠）
GMV目標: 73.6億円

ROAS = 73.6億 ÷ 15.8億 = 466%

別枠（ジョングク2.5-3.3億）を含めた場合:
ROAS = 73.6億 ÷ 18.3億 = 402%

どちらも高ROASを維持
```
