# Q1計画の根拠 (Q1 Plan Reasoning)

q1.md の投資配分・施策選定の根拠・計算式を説明する。

---

## 1. Q1投資3.8億円の配分根拠

### 1.1 月別配分（2月 vs 3月）の理由

**配分構成**:

| 月 | 予算 | 比率 | 役割 |
|----|------|------|------|
| 2月 | 0.8億円 | 21% | 立ち上げ・準備期間 |
| 3月 | 3.0億円 | 79% | メガ割全力期 |

**ロジック**:
```
メガ割売上の70-80%は開催期間中に発生
→ 3月に投資を集中させるのが最も効率的

2月は最小限の投資で:
- 認知の下地を作る
- 運用体制を構築する
- KOL/インフルエンサーとの関係構築
```

---

### 1.2 Ambassador無しの決定根拠

**判断プロセス**:

| 選択肢 | メリット | デメリット |
|--------|----------|------------|
| Q1でAmbassador起用 | 認知一気に獲得 | 準備期間不足、費用対効果不明 |
| **Q1はAmbassador無し** | 6月ジョングクに集中、予算効率化 | Q1の認知力は限定的 |

**決定理由**:
1. **準備期間不足**: 2月15日開始では調整が間に合わない
2. **6月ジョングクへの集中**: 別枠予算を確保し最大効果を狙う
3. **Q1は「テスト期間」**: パフォーマンス施策の効果検証

**競合事例からの学び**:
- Dalba: Ambassador（三吉彩花）起用はあったが、メガ割成功の主因はMocchi 80投稿
- メガ割はAmbassadorより「Xハック + KOL Pick」が効く

---

### 1.3 媒体別予算配分の根拠

**配分構成**:

| 媒体 | 予算 | 比率 | 配分理由 |
|------|------|------|---------|
| TikTok（制作+GMV MAX） | 1.00億 | 26% | 認知の絶対量を確保 |
| X RT部隊 | 0.70億 | 18% | 美容カテゴリのXハックが勝ち筋 |
| IG KOL Pick | 0.40億 | 11% | メガ割GMVの40-50%はKOL経由 |
| TTS GMV MAX | 0.40億 | 11% | EC直接売上・学習期間確保 |
| X 運用型広告 | 0.20億 | 5% | 補完的リーチ拡大 |
| YT メガ系KOL | 0.15億 | 4% | 刈り取り施策 |
| その他 | 0.95億 | 25% | シーディング、@cosme、予備 |

**ロジック**:
```
TikTok + X（RT部隊）= 認知形成の柱（44%）
KOL Pick（IG/YT）= 刈り取りの柱（15%）
TTS GMV MAX = 常時売上確保（11%）
その他 = 補完・予備（30%）
```

## 2. TikTok 1,300本の算出根拠

### 2.1 本数目標の設定

**計算式**:

```
競合上位ブランド: 月500-1,000本投稿
Dr.Melaxin目標: 競合同等以上

Q1配分:
- 2月（立ち上げ）: 300本（1.5週間準備、2週間で150本×2）
- 3月（メガ割）: 1,000本（週250本ペース）
- 合計: 1,300本
```

**週別スケジュール根拠**:

| 期間 | 本数 | 理由 |
|------|------|------|
| 2/15-21 | 150本 | 立ち上げ、認知の種まき |
| 2/22-28 | 150本 | 加速、UGC醸成 |
| 3/1-5 | 200本 | メガ割直前、バズ狙い |
| 3/6-10 | 200本 | メガ割開始、購入誘導 |
| 3/11-15 | 150本 | 中盤、追い上げ |
| 3/16-20 | 150本 | 終盤、ラストスパート |
| 3/21-31 | 300本 | 継続、Q2への布石 |

### 2.2 制作費用の計算

**計算式**:

```
制作単価: ¥10,000/本（Jipanglobal）
1,300本 × ¥10,000 = 1,300万円

GMV MAX配信費:
TikTok総予算1.0億円 - 制作費1,300万円 = 8,700万円

ROAS 200%想定:
8,700万円 × 200% = 1.74億円 GMV（TikTok Shop経由）
```

### 2.3 コンテンツタイプ配分の根拠

| タイプ | 比率 | 理由 |
|--------|------|------|
| 話題系 | 30% | バズりやすい、認知獲得 |
| 検証系 | 25% | 信頼性、購買検討層向け |
| B/A | 20% | 効果実感、CVR高い |
| 購入誘導 | 15% | メガ割期間中の刈り取り |
| バズ狙い | 10% | ハイリスク・ハイリターン |

## 3. RT部隊 0.7億円の設計根拠

### 3.1 投資規模の算出

**計算式**:

```
imp単価: ¥1（業者実績）
予算: 0.7億円
→ 総imp: 7,000万imp

freq（平均接触回数）: 4回
→ Reach: 7,000万 ÷ 4 = 1,750万UU
```

### 3.2 アカウント設計の根拠

**構成**:

| ペルソナ | アカウント数 | フォロワー計 | 役割 |
|---------|------------|-------------|------|
| バズコス暴露ちゃん | 4-5 | 40-50万 | 話題化、拡散起点 |
| 顔面工事ちゃん | 4-5 | 40-50万 | B/A訴求、整形垢層リーチ |
| 成分解析オタク | 3-4 | 30-40万 | 信頼性、検証系 |
| コスメオタク | 2-3 | 20-30万 | ファン形成、リピート |
| 一般ユーザー風 | 2-3 | 20-30万 | 自然なUGC、信頼性 |

**ペルソナ分散の理由**:
1. **垢バレリスク軽減**: 1ペルソナ露出で全滅しない
2. **ターゲット層の多様性**: 美容感度高い層〜一般層まで網羅
3. **コンテンツの自然さ**: 同一ペルソナばかりだと不自然

### 3.3 投稿スケジュールの根拠

| 期間 | 投稿数/日 | 理由 |
|------|----------|------|
| 2/15-28 | 5-10投稿 | 助走期間、話題の種まき |
| 3/1-5 | 10投稿 | 直前期、期待感醸成 |
| 3/6-15 | 15-20投稿 | 全力期、トレンド入り狙い |
| 3/16-20 | 20投稿 | ラストスパート |

**トレンド入り目標「5回以上」の根拠**:
- LAKA: RT部隊で月2-3回のトレンド入り
- 予算規模（0.7億円）から5回以上は達成可能と判断

## 4. KOL Pick 0.4億円の配分根拠

### 4.1 Aクラス2人契約の理由

**KOL規模と効果**:

| Tier | フォロワー | 配信GMV目安 | 単価 |
|------|-----------|-------------|------|
| Aクラス | 10-30万 | 3,000-5,000万円/配信 | 500-1,000万円 |
| Bクラス | 5-10万 | 500-1,000万円/配信 | 100-300万円 |

**2人契約の根拠**:
```
Q1目標GMV: 5-7億円
KOL Pick経由比率: 40-50%
→ KOL Pick目標GMV: 2-3.5億円

Aクラス2人 × 3配信 = 6配信
6配信 × 3,000-5,000万円 = 1.8-3.0億円GMV

→ 目標達成には十分な規模
```

### 4.2 配信スケジュールの根拠

| 日程 | KOL | 目標GMV | タイミング理由 |
|------|-----|---------|---------------|
| 3/3 | Aクラス① | 3,000万 | メガ割直前、期待感醸成 |
| 3/7 | Aクラス② | 5,000万 | メガ割2日目、勢いづけ |
| 3/10 | Aクラス① | 4,000万 | 中盤、追加刈り取り |
| 3/15 | Bクラス×2 | 2,000万 | 後半、幅広い層へ |
| 3/18 | Aクラス② | 3,000万 | 終盤、最終プッシュ |
| 3/20 | 全員 | 3,000万 | 最終日、総力戦 |

**ロジック**:
- メガ割序盤（3/3-7）に主力投入 → 勢いをつける
- 中盤（3/10-15）で維持 → ランキング安定
- 終盤（3/18-20）で追い込み → 最終順位確定

## 5. GMV目標5-7億円の算出根拠

### 5.1 新規参入ブランドの実績ベンチマーク

**競合の初回メガ割実績**:

| ブランド | 初回メガ割 | 備考 |
|----------|-----------|------|
| Lower Tier平均 | 3-5億円 | BOH, Numberzin等 |
| Middle Tier初回 | 5-8億円 | 一定の認知がある場合 |

**Dr.Melaxinの条件**:
- 日本市場での認知: ほぼゼロ（新規参入）
- 投資規模: 3.8億円（競合比で大きい）
- 施策: TikTok大量生成 + RT部隊 + KOL Pick

**目標設定ロジック**:
```
新規参入 → Lower Tier相当（3-5億円）がベースライン
大量投資 → Middle Tier相当（5-7億円）を目指す

5億円（下限）: 最低限達成すべきライン
7億円（上限）: 施策が全てうまくいった場合
```

### 5.2 チャネル別GMV内訳

| チャネル | GMV | 算出根拠 |
|---------|-----|---------|
| Qoo10 | 4.0-5.5億円 | KOL Pick + オーガニック |
| TikTok Shop | 0.8億円 | GMV MAX ROAS 200% |
| Amazon | 0.2億円 | 検索流入 |
| 楽天 | 0.1億円 | 最小投資 |
| **合計** | **5.1-6.6億円** | |

### 5.3 ROAS 130-180%の根拠

**計算式**:

```
投資額: 3.8億円
GMV目標: 5-7億円

下限ROAS: 5億 ÷ 3.8億 = 132%
上限ROAS: 7億 ÷ 3.8億 = 184%

→ 目標ROAS: 130-180%
```

**注意**: Q1は「投資フェーズ」であり、年間ROAS 466%を達成するための布石。Q1単体でのROASは低くても許容。

## 7. リスク分析と対策の根拠

### 7.1 リスク評価

| リスク | 確率 | 影響度 | スコア | 対策 |
|--------|------|--------|--------|------|
| RT部隊垢バレ | 中 | 高 | 高 | ペルソナ分散、即削除 |
| 在庫切れ | 中 | 高 | 高 | 事前大量確保 |
| 競合激化 | 高 | 中 | 中 | RT部隊強化、価格調整 |
| TikTok本数不足 | 低 | 中 | 低 | Jipanglobal追加発注 |
| KOL Pick不調 | 低 | 中 | 低 | Bクラス追加投入 |

### 7.2 RT部隊リスク対策の詳細

**垢バレ・炎上の対策**:

| 対策 | 内容 | 根拠 |
|------|------|------|
| ペルソナ分散 | 5種類のペルソナで分散 | 1ペルソナ露出で全滅しない |
| 即時削除 | 炎上投稿は即削除 | 過去事例で被害最小化に有効 |
| 名前変更 | 必要に応じてアカウント名変更 | 継続運用のベストプラクティス |
| 間接的言及 | 商品名を直接言及しない | ステマ指摘リスク軽減 |

**過去事例からの学び**:
- 競合（LAKA, Medicube, VT等）は全てRT部隊を活用
- 炎上しても名前変えて継続している業者が多い
- 完全防止は不可能 → リスク軽減で対応

### 7.3 在庫切れリスク対策

**対策**:

| 対策 | 内容 |
|------|------|
| 事前大量確保 | GMV目標の150%相当を在庫確保 |
| 追加発注体制 | 韓国本社との連携で緊急発注可能に |
| 売り切れ施策 | 「売り切れ注意」訴求でむしろ購買促進 |

**競合の在庫切れ事例**:
- 一部ブランドはメガ割中に在庫切れでランキング下落
- 機会損失だけでなくブランドイメージにも影響

## 8. 2月の役割（投資0.8億円）の根拠

### 8.1 「立ち上げ期」の位置づけ

**2月の目的**:

| 項目 | 内容 |
|------|------|
| 認知の種まき | TikTok 300本で話題の下地を作る |
| 運用体制構築 | TTS GMV MAX、RT部隊の運用開始 |
| KOL関係構築 | シーディング100名でUGC醸成 |
| 3月メガ割準備 | 素材制作、配信スケジュール確定 |

**投資を抑える理由**:
```
2月は売上を作る期間ではない
→ 認知蓄積と運用テストが目的
→ 最小限の投資で効果を検証
→ 3月メガ割に向けた準備期間
```

### 8.2 2月施策の詳細根拠

| 施策 | 予算 | 目的 |
|------|------|------|
| TikTok 300本 | 0.30億 | 認知の下地、アルゴリズム学習 |
| RT部隊開始 | 0.20億 | フォロワー獲得、運用テスト |
| KOLシーディング | 0.15億 | UGC醸成、口コミ形成 |
| TTS GMV MAX | 0.10億 | 運用体制構築、学習期間確保 |
