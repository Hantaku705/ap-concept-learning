#### 概要

| 項目 | 値 |

| 3/6-10 | 200本 | 購入誘導系（「今買わないと」） |
| 3/11-15 | 150本 | 追い上げ系（「売り切れ注意」） |
| 3/16-20 | 150本 | 最終系（「ラストチャンス」） |
| **合計** | **1,000本** | |

#### コンテンツタイプ

| タイプ | 比率 | 内容 |
