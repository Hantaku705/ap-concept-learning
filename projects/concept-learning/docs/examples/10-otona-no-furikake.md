## コンセプト分析

### 既存の概念

> 子供向けのご飯のお供

### 新しい視点

> 大人も楽しめる本格的な味

### 顧客インサイト

> 「ふりかけ食べたいけど、子供っぽいと思われたくない」

### コンセプト

> **「大人のふりかけ」**

## 公式に当てはめると

```
（ふりかけ）を（大人向け）の視点で捉え直し、

```

## なぜ良いコンセプトか

### リフレーミング
- ふりかけを「子供向け」から「大人の贅沢」に再定義
- 年齢制限を外すだけで新市場

### 顧客インサイト
- 大人になってもふりかけを楽しみたい潜在ニーズ
- 許可を与えるコンセプト

### シンプルで直感的
- 「大人の」で全てが伝わる
- ターゲットが明確

### 行動を生み出す
- 「大人も食べていいんだ」と安心して買える
- 堂々と楽しめる

## ネーミング案

1. **本格派ふりかけ** - 品質+商品が明確
2. **贅沢ごはんの友** - 感情+商品で価値訴求
3. **大人味トッピング** - 対象+商品を表現
4. **リッチふりかけ** - 品質+商品でプレミアム感
5. **プレミアムご飯のお供** - 品質+商品で高級路線
