## 1. Q3投資4.0億円の配分根拠

### 1.1 月別配分の理由

**配分構成**:

| 月 | 予算 | 比率 | 役割 |
|----|------|------|------|
| 7月 | 1.0億円 | 25% | TTS強化期 |
| 8月 | 1.0億円 | 25% | TTS強化期・準備 |
| 9月 | 2.0億円 | 50% | メガ割 + Drug Store参入 |

**ロジック**:
```
Q3の特徴:
1. 7-8月はEC閑散期 → 認知蓄積とTTS強化に集中
2. 9月にダブルイベント → メガ割 + Drug Store参入
3. 投資は9月に集中（年間パターンを踏襲）
```

### 1.2 Q1-Q2との比較

| 四半期 | 投資額 | メガ割投資 | 特徴 |
|--------|--------|-----------|------|
| Q1 | 3.8億円 | 3.0億円 | 立ち上げ、Ambassador無し |
| Q2 | 3.6億円+別枠 | 2.0億円 | ジョングク効果 |
| **Q3** | **4.0億円** | **2.0億円** | Drug Store参入、TTS強化 |

**Q3が最大投資となる理由**:
1. **Drug Store参入費用**: 0.5億円が追加
2. **TTS強化**: 7-8月の日予算増額
3. **2チャネル同時運営**: EC + 店頭のオペレーション

## 2. Drug Store参入時期（9月）の根拠

### 2.1 9月参入の理由

**タイミング選定の論点**:

| 選択肢 | メリット | デメリット |
|--------|----------|------------|
| Q2（6月） | ジョングクと同時 | 準備期間不足、リソース分散 |
| **Q3（9月）** | 認知確立後、安定運営 | ジョングク効果は間接的 |
| Q4（11月） | 年末商戦に間に合う | 参入が遅すぎる |

**9月選定の根拠**:
```
1. 認知の蓄積:
   - Q1-Q2でEC中心の認知を確立
   - 「Qoo10で話題のブランド」としての実績

2. オペレーション安定:
   - Q2のジョングクイベントが終了
   - リソースをDrug Store対応に集中可能

3. 秋の購買シーズン:
   - 9-11月は美容商材の購買が活発化
   - 年末に向けた棚確保が可能

4. メガ割との相乗効果:
   - 「店頭でも買える」訴求がメガ割を後押し
   - EC→店頭、店頭→ECの相互送客
```

### 2.2 パートナー選定の根拠

**候補比較**:

| チェーン | 店舗数 | 強み | 適合度 |
|----------|--------|------|--------|
| **マツモトキヨシ** | 1,700店 | 韓国コスメ専門売場 | ★★★★★ |
| **ウエルシア** | 2,000店+ | 店舗数最大 | ★★★★☆ |
| ツルハ | 2,500店 | 北海道・東北強い | ★★★☆☆ |
| ココカラファイン | 1,300店 | マツキヨと統合中 | ★★★★☆ |

**マツモトキヨシ優先の理由**:
1. **韓国コスメ専門コーナー**: 既存の売場があり、導入しやすい
2. **若年層来店**: ターゲット層（10-30代女性）の来店率が高い
3. **ブランドイメージ**: 「マツキヨにある」は信頼感につながる

### 2.3 初期店舗数500-1,000店の根拠

**計算式**:

```
総店舗数: 主要3チェーンで約6,000店舗
初年度目標カバー率: 10-15%
→ 初期店舗数: 600-900店舗 ≒ 500-1,000店舗

段階的拡大計画:
- 9月: 500店舗（主要都市）
- 10月: +300店舗（地方都市）
- 11月: +200店舗（その他）
- 年末: 1,000店舗達成
```

| 時期 | Qoo10 | TikTok Shop |

| メガ割月 | ★★★★★ | ★★★☆☆ |

**Q1-Q2との比較**:

| 時期 | 日予算 | 理由 |

| Q2（4-5月） | 10-30万/日 | 維持モード |
| **Q3（7-8月）** | **20-40万/日** | 強化期間 |

**増額の根拠**:
```

4. Q3 GMV 4億円目標:

## 4. GMV目標8億円の算出根拠

### 4.1 Q2（10億円）比80%の理由

**成長ドライバーの変化**:

| 要因 | Q2 | Q3 | 変化 |
|------|-----|-----|------|
| ジョングク効果 | 最大 | 継続（素材再利用） | ↓ |
| 認知の蓄積 | 高 | さらに蓄積 | ↑ |
| Drug Store | なし | 参入 | ↑ |
| 季節要因 | 初夏 | 秋 | → |

**計算式**:

```
Q2 GMV: 10億円
基本成長: 認知蓄積で+10% → 11億円
ジョングク効果減少: -30% → 7.7億円
Drug Store参入: +5% → 8.1億円

→ 目標GMV: 8億円（保守的）
```

### 4.2 チャネル別GMV内訳

| チャネル | GMV | 算出根拠 |
|---------|-----|---------|
| Qoo10 | 5.0億円 | KOL Pick + 認知継続 |
| TikTok Shop | 1.5億円 | GMV MAX強化 |
| Amazon | 0.5億円 | タイムセール連動 |
| 楽天 | 0.2億円 | 最小投資 |
| Drug Store | 0.8億円 | 初月1億円目標の80% |
| **合計** | **8.0億円** | |

### 4.3 ROAS 200%の根拠

**計算式**:

```
投資額: 4.0億円
GMV目標: 8.0億円

ROAS = 8.0億 ÷ 4.0億 = 200%

Q1-Q2との比較:
- Q1: 130-180%（立ち上げ）
- Q2: 280%（ジョングク効果）
- Q3: 200%（安定運用）

→ Q3は「安定運用」フェーズで200%が妥当
```

## 5. オン×オフ連携戦略の根拠

### 5.1 連携施策の設計理由

**課題認識**:
```
EC購入者 ≠ 店頭購入者
→ 相互送客で両チャネルを伸ばす

ECユーザーの特徴:
- 価格比較、レビュー重視
- 「試してから買いたい」ニーズ

店頭ユーザーの特徴:
- 衝動買い、すぐ欲しい
- 「限定品」「セット」に弱い
```

### 5.2 施策別の根拠

| 施策 | 予算 | 目的 | 効果 |
|------|------|------|------|
| TikTok店頭投稿 | 制作費内 | EC→店頭送客 | 店頭認知UP |
| RT部隊「店頭発見」系 | 0.1億 | 話題化 | トレンド入り |
| KOL「店頭購入レビュー」 | 0.05億 | 信頼性 | 購買意欲UP |
| **EC限定セット** | - | **差別化** | **EC誘導** |

**EC限定セットの設計理由**:
```
問題: 店頭と同じ商品だとEC優位性がない
解決: EC限定セット（増量、限定品付き）で差別化

例:
- 店頭: Serum単品 ¥3,980
- EC: Serum + ミニサイズ + ポーチ ¥4,980（実質¥1,500お得）

→ 店頭で興味を持った人が「ECだとお得」と認識
→ 店頭→EC誘導に成功
```

店舗あたり月販売数: 40個
客単価: ¥5,000

月間売上 = 500店 × 40個 × ¥5,000 = 1.0億円

| 項目 | 設定値 | 根拠 |
|------|--------|------|
| 店舗数 | 500店 | 主要都市の主要店舗 |
| 店舗あたり販売数 | 40個/月 | 競合実績から推定 |
| 客単価 | ¥5,000 | Serum + α |

**競合ベンチマーク**:
```
韓国コスメ Drug Store売上（推定）:
- Top Tier: 月2-3億円（1,000店以上）
- Middle Tier: 月0.5-1.0億円（500-1,000店）

Dr.Melaxin（新規参入）:
- 500店舗で1億円/月は挑戦的だが達成可能

## 9. リスク分析と対策の根拠

### 9.1 Drug Store参入リスク

| リスク | 確率 | 影響度 | 対策 |
|--------|------|--------|------|
| 参入遅延 | 中 | 高 | Q4への延期も想定 |

**参入遅延の対策詳細**:
```

1. 契約交渉の長期化

対策:
- Q4への延期も計画に組み込み
- 延期時は9月のEC投資を強化

```
