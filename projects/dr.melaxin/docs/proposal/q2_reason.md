# Q2計画の根拠 (Q2 Plan Reasoning)

q2.md の投資配分・ジョングク起用・オフラインイベント設計の根拠・計算式を説明する。

---

## 1. Q2投資配分の根拠

### 1.1 二重予算構造の理由

**構成**:

| カテゴリ | 予算 | 比率 | 役割 |
|---------|------|------|------|
| $10M枠 | 3.6億円 | 52-59% | パフォーマンス施策 |
| ジョングク別枠 | 2.5-3.3億円 | 41-48% | 認知爆発施策 |
| **合計** | **6.1-6.9億円** | 100% | Q2総投資 |

**別枠化の理由**:
1. **$10M枠を圧迫しない**: パフォーマンス施策の予算を確保
2. **効果測定の明確化**: ジョングク効果を単独で測定可能
3. **経営判断の分離**: Ambassador起用は戦略的判断として別管理

### 1.2 月別配分の根拠

**$10M枠の配分**:

| 月 | 予算 | 比率 | 役割 |
|----|------|------|------|
| 4月 | 0.8億円 | 22% | 継続・準備期間 |
| 5月 | 0.8億円 | 22% | 準備・リーク期間 |
| 6月 | 2.0億円 | 56% | メガ割全力期 |

**ロジック**:
```
Q1の成功パターン（通常期→メガ割）を継続
4-5月: 認知維持、6月準備に注力
6月: メガ割 + ジョングク効果で最大化

6月集中投資の理由:
- メガ割売上の70-80%が開催期間中
- ジョングク発表との相乗効果
- Q1比較で150%成長を狙う
```

## 2. ジョングク起用の決定根拠

### 2.1 Big KOL選定の理由

**候補比較**:

| 候補 | 認知度 | ファンベース | コスト | 適合度 |
|------|--------|-------------|--------|--------|
| **ジョングク** | 最高 | 巨大（BTS） | 高 | ★★★★★ |
| SEVENTEEN | 高 | 大 | 中 | ★★★★☆ |
| Stray Kids | 高 | 大 | 中 | ★★★★☆ |
| 国内タレント | 中 | 国内限定 | 中 | ★★★☆☆ |

**ジョングク選定の根拠**:
1. **日本でのファンベース巨大**: BTSメンバー = 数百万人のファン
2. **韓国コスメとの親和性**: K-POPアイドル × 韓国コスメは黄金組み合わせ
3. **競合との差別化**: Dalba（三吉彩花）を上回るインパクト
4. **話題性**: 発表だけでトレンド入り確実

### 2.2 競合Ambassador比較

| ブランド | Ambassador | 効果 |
|----------|-----------|------|
| Dalba | 三吉彩花（中堅A級） | メガ割2位達成に寄与 |
| Medicube | タレント起用 | 認知拡大に成功 |
| **Dr.Melaxin** | **ジョングク（トップ級）** | **競合を上回るインパクト** |

**差別化のロジック**:
```
Dalba: 三吉彩花（中堅A級）で成功
→ Dr.Melaxin: トップ級Ambassador（ジョングク）で更なる成功を狙う

中堅A級 vs トップ級の差:
- 認知リーチ: 3-5倍
- ファンの熱量: BTSファンの購買力は極めて高い
- メディア露出: 自然発生するPR効果
```

### 2.3 予算別枠2.5-3.3億円の内訳根拠

**計算式**:

| 項目 | 予算 | 算出根拠 |
|------|------|---------|
| ギャランティ | 1.5-2.0億 | K-POPトップ級の年間契約相場 |
| 撮影・制作 | 0.3億 | TV CM 15秒/30秒 + SNS素材 |
| オフラインイベント | 0.5億 | ポップアップストア2週間 |
| PR・メディア露出 | 0.2-0.5億 | プレス発表、メディア招待 |
| **合計** | **2.5-3.3億円** | |

**ギャランティ算出**:
```
トップK-POPアイドルの日本市場相場:
- 単発キャンペーン: 0.5-1.0億円
- 年間契約: 1.5-3.0億円

ジョングク（BTSメンバー）:
- 市場価値: 最高クラス
- 想定年間契約: 1.5-2.0億円（複数回利用可）
```

## 3. オフラインイベント設計の根拠

### 3.1 ポップアップストア選定の理由

**選択肢比較**:

| 形態 | メリット | デメリット | コスト |
|------|----------|------------|--------|
| **ポップアップ** | 体験提供、SNS映え、直接販売 | 期間限定 | 中 |
| ショールーム | 常設、ブランド拠点 | 来場数限定 | 高 |
| イベント | 短期集中、話題性 | 持続性なし | 中 |

**ポップアップ選定の根拠**:
1. **SNS映え**: フォトスポット、ジョングクビジュアルで投稿誘発
2. **体験提供**: 商品テスター、肌診断で購買検討を促進
3. **限定感**: 期間限定で来場動機を強化
4. **EC連動**: 店頭で興味喚起 → Qoo10メガ割で購入

### 3.2 来場目標1万人の算出

**計算式**:

```
開催期間: 14日間
1日あたり目標: 1万人 ÷ 14日 = 約714人/日

平日: 500人/日 × 10日 = 5,000人
土日: 1,250人/日 × 4日 = 5,000人
合計: 10,000人

ベンチマーク:
- 韓国コスメポップアップ（渋谷/原宿）: 5,000-15,000人
- アイドルコラボイベント: 10,000-30,000人
→ ジョングク効果で1万人は達成可能
```

### 3.3 SNS投稿5,000件の算出

**計算式**:

```
来場者: 10,000人
投稿率: 50%（フォトスポットあり、特典あり）
→ 投稿数: 10,000 × 50% = 5,000件

投稿からのリーチ:
- 1投稿あたり平均フォロワー: 500人
- 視聴率: 20%
- 5,000件 × 500人 × 20% = 500万リーチ
```

### 3.4 会場選定の根拠

**候補エリア**: 渋谷/原宿

| 会場 | 坪数 | 特徴 | 適合度 |
|------|------|------|--------|
| ラフォーレ原宿 | 50-100坪 | 若年層、ファッション系 | ★★★★★ |
| @cosme TOKYO | 100坪+ | 美容特化、購買導線 | ★★★★★ |
| SHIBUYA 109 | 30-50坪 | 認知度高い、若年層 | ★★★★☆ |
| 表参道ヒルズ | 50-100坪 | 高級感、30代以上 | ★★★☆☆ |

**選定理由**:
- 10-30代女性が集まるエリア = ターゲット層と一致
- 韓国コスメに親和性のある立地
- SNS映えする空間設計が可能

## 4. GMV目標10億円の算出根拠

### 4.1 Q1比150%成長の理由

**成長ドライバー**:

| 要因 | Q1 → Q2の変化 | GMV効果 |
|------|--------------|---------|
| ジョングク効果 | なし → あり | +30-50% |
| 認知の蓄積 | 立ち上げ → 2回目 | +20% |
| KOL Pick強化 | 4-6配信 → 6-8配信 | +10% |
| 季節要因 | 春 → 初夏 | 中立 |

**計算式**:

```
Q1 GMV: 5-7億円（中央値6億円）
成長率: 150-170%

基本成長（認知蓄積）: +20% → 7.2億円
ジョングク効果: +40% → 10.1億円

→ 目標GMV: 10億円
```

### 4.2 チャネル別GMV内訳

| チャネル | GMV | 算出根拠 |
|---------|-----|---------|
| Qoo10 | 8.0億円 | KOL Pick + ジョングク効果 |
| TikTok Shop | 1.0億円 | GMV MAX ROAS 200% |
| Amazon | 0.5億円 | 検索流入増加 |
| 楽天 | 0.2億円 | 最小投資 |
| オフライン | 0.3億円 | ポップアップ直販 |
| **合計** | **10.0億円** | |

### 4.3 ROAS 280%（$10M枠のみ）の根拠

**計算式**:

```
$10M枠投資: 3.6億円
GMV目標: 10億円

ROAS = 10億 ÷ 3.6億 = 278% ≒ 280%

注意:
- ジョングク別枠（2.5-3.3億円）を含めると:
  ROAS = 10億 ÷ 6.1億 = 164%
- $10M枠のみで計算するのは、パフォーマンス施策の効率を測るため
```

## 8. リスク分析と対策の根拠

### 8.1 ジョングク関連リスク

| リスク | 確率 | 影響度 | 対策 | 対策の根拠 |
|--------|------|--------|------|-----------|
| 契約不可 | 中 | 高 | Plan B発動 | 4月末判断で6月準備に2ヶ月確保 |
| スキャンダル | 低 | 最高 | 契約条項、素材撮り溜め | 事前に多量素材を確保 |
| 日程変更 | 中 | 中 | 柔軟なスケジュール | イベント日程を調整可能に設計 |

**スキャンダル対策の詳細**:
```
1. 契約条項に明記:
   - スキャンダル発生時の対応ルール
   - 素材使用の継続可否判断基準

2. 素材撮り溜め:
   - 年間分の素材を初回撮影で確保
   - 複数バリエーションを制作
   - スキャンダル時も既存素材で継続可能

3. PR戦略:
   - ジョングク単独ではなくブランド主体の訴求
   - 商品効果中心のコミュニケーション
```

### 8.2 イベント関連リスク

| リスク | 確率 | 対策 |
|--------|------|------|
| 来場者不足 | 低 | SNS告知強化、KOL来店イベント |
| 混雑 | 高 | 整理券制、時間指定、Web予約 |
| 天候不良 | 中 | 室内会場選定（ラフォーレ等） |
| 在庫切れ | 中 | 事前大量確保、追加発注体制 |

**混雑対策の詳細**:
```
予想:
- ジョングク効果で来場集中の可能性
- 特に土日は1,000人超/日の可能性

対策:
1. 整理券制度:
   - 当日配布 + Web事前予約
   - 時間帯別に枠設定

2. 滞在時間管理:
   - 目安滞在時間: 30-60分
   - 回転率: 20-40人/時間

3. オンライン購入誘導:
   - 店頭は体験、購入はQoo10へ
   - QRコードでメガ割ページ誘導
```

| 項目 | Q1（2月） | Q2（4-5月） |

**維持モードの理由**:
```
目的: 認知の維持と6月メガ割の準備

具体的には:

3. リーク施策: 6月への期待感醸成
```
