## 採用・不採用の理由

### SP採用理由

1. **圧倒的リーチ**: 90M Imp、Reach 34.2Mで話題化の核
2. **CPM効率**: ¥300で3種の中で最も効率的
3. **一番槍**: 2/1花粉速報時にトレンド入り確実

### TL/TLR不採用理由

- デモグラ設定不可 → 男性にも当たる
- アンプリファイ不可 → 同予算なら運用型の方が効率的
