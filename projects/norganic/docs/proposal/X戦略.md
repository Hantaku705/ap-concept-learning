## 戦略方針

### TL/TLR不採用の理由
- デモグラ設定不可 → 男性にも当たる
- アンプリファイ不可 → 同予算なら運用型の方が効率的

- **CPM ¥300** で3種の中で最も効率的
- **90M Imp、Reach 34.2M** で圧倒的なリーチ
- 2/1花粉速報時に「一番槍」を取れる

| **1位** | 話題化 | スポットライトTKO | 90M Impで第一想起獲得 |

## FQ試算

### 前提条件

| 項目 | 数値 |
|------|------|
| ターゲット規模（30代女性×X利用） | 約400万人 |
| 運用型広告CPM | 約400円 |
| ターゲティング可能予算 | 2,300万円（SP除く） |

### Imp試算

| 施策 | 予算 | Imp | 備考 |
|------|------|-----|------|
| スポットライトTKO | 2,700万円 | 90M | デモグラ設定不可 |

### FQ計算

```
ターゲティング可能Imp: 6,500万
FQ = 6,500万Imp ÷ 400万人 = 約16回
```

**→ FQ5以上は十分達成可能**
