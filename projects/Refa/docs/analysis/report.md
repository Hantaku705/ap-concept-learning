# ReFa プロモーション変遷分析レポート
## 2019年〜2025年：インバウンド崩壊からの復活

**作成日**: 2026年1月26日
**対象**: MTG株式会社 美容ブランド「ReFa」

---

## 1. エグゼクティブサマリー

### ReFaの軌跡

ReFaは2009年に美顔ローラーから始まり、2018年に累計1,000万本を突破。しかし2019年、中国新EC法施行によるインバウンド崩壊で**営業利益-144億円の赤字**に転落。

その後、製品多角化・チャネル戦略・プロモーション手法の大転換により、2025年には**売上728億円（+435%）**・過去最高を達成した。

### 主要な発見

| 指標 | 2019年（危機） | 2025年（最新） | 変化 |
|------|---------------|---------------|------|
| MTG売上高 | 360億円 | 988億円 | **+175%** |
| 営業利益 | -144億円 | 107億円 | **+251億円** |
| ReFa売上 | 136億円 | 728億円 | **+435%** |
| MTG内比率 | 38% | 74% | +36pt |

## 3. 時系列分析：5つのフェーズ

### Phase 1: ローラー時代（2009年〜2018年）

**概要**: エステティシャンの手技を再現した美顔ローラーで市場を創造

#### 製品
- ReFa CARAT（フラッグシップ、2-3万円）
- ReFa S CARAT（パーツケア用、2014年発売）
- ReFa GRACE HEAD SPA（頭皮用、2016年発売）

#### プロモーション手法
| 手法 | 内容 |
|------|------|
| 口コミ・PR | 芸能人の自発的使用（IKKOなど）、美容雑誌掲載 |
| 受賞 | @cosmeベストコスメアワード2016殿堂入り |
| 体験導入 | エステサロン・美容室での設置 |
| 海外展開 | 2017年、上海に海外初の単独カウンター |

#### 実績
- 2014年: 累計300万本突破
- 2017年: 累計700万本突破
- 2018年: **累計1,000万本突破**

#### この時期の特徴
- **単一カテゴリ集中**: 美顔ローラーに特化
- **インバウンド依存**: 中国代購（代理購入業者）が売上の大部分を占める
- **プロモーションは受動的**: 芸能人の自発使用・口コミ中心

---

### Phase 2: インバウンド崩壊と戦略転換（2019年）

**概要**: 外部環境の激変により過去最大の危機。同時に大転換の起点となった年。

#### 危機の要因

| 要因 | 影響 |
|------|------|
| 中国新EC法施行（1月） | 代購激減（142億円減少） |
| 韓国不買運動 | 韓国市場縮小 |
| 香港デモ | アジア市場不安定化 |
| **合計影響** | **売上155億円減少** |

#### 業績影響
- 売上高: 584億円 → 360億円（**-38%**）
- 営業利益: 92億円 → **-144億円**（赤字転落）
- ReFa売上の191億円減少のうち、142億円がインバウンド要因

#### 戦略転換の決断

**2019年10月: ReFa BEAUTECHシリーズ発売**

| 製品 | 価格 | 特徴 |
|------|------|------|
| ReFa BEAUTECH DRYER | 36,300円 | プロセンシング、ハイドロイオン |
| ReFa BEAUTECH STRAIGHT IRON | 22,000円 | カーボンレイヤープレート |

**重要な戦略判断**:
- 美容室20以上と共同開発（「レア髪」コンセプト）
- **最初は美容室限定販売**（ブランドイメージ構築優先）
- 「家電量販店で売っているドライヤー」ではなく「美容室で売っているドライヤー」というポジショニング

*出典: [MTG News - BEAUTECH発売](https://www.mtg.gr.jp/news/detail/2019/10/article_1816.html)*

### Phase 3: 製品多角化・国内市場強化（2020年〜2021年）

**概要**: コロナ禍を追い風に、新カテゴリ製品で国内市場を開拓

#### 製品拡大

| 時期 | 製品 | 価格 | 結果 |
|------|------|------|------|
| 2020年4月 | ドライヤー・アイロン販路拡大 | - | EC、百貨店、一部家電量販店へ |

| 2021年 | ヘアケアラインナップ拡充 | - | カールアイロン等追加 |

#### シャワーヘッド成功の背景
- コロナ禍で在宅時間増加
- 清潔志向の高まり
- 「自宅でサロン体験」ニーズにマッチ

#### プロモーション手法の変化

| 従来（〜2019年） | 新戦略（2020年〜） |
|-----------------|-------------------|
| 口コミ・PR依存 | プロユース訴求 |
| 芸能人の自発使用 | 美容師との共同開発ストーリー |
| インバウンド重視 | 国内EC強化 |

**新コンセプト**: 「サロン帰りの"レア髪"を自宅で再現」

#### 実績
- ReFa売上: 136億円（2019年） → **219億円**（2020年、**+61%**）
- 国内EC売上: 大幅増加
- 美容室での好評: 一部店舗で入荷直後に欠品状態

#### 再ブランディングへの転換：ReFa GINZA

FY2025の好業績を受け、MTGは「収穫期」に入るのではなく、**再びブランディング強化に舵を切った**。その象徴がReFa GINZAへの大型投資である。

**ReFa GINZA 旗艦店**

| 項目 | 内容 |
|------|------|
| オープン | **2025年11月15日** |
| 立地 | 銀座（詳細住所非公開） |
| コンセプト | **"ReFa Amazing Luxury"** |
| ターゲット | 10代から3世代、3,000円〜数十万円の幅広い商品 |

**クリエイティブ投資**:
- **GWENAEL NICOLAS**（建築家）等、世界的クリエイターとのコラボレーション
- 「訪れた全ての人の心が躍る、賑やかなラグジュアリー」をテーマにした空間設計

**ブランド成長戦略サイクル**:

MTGは4つのサイクルでブランドを成長させる戦略を採用：

```
① ブランド価値の向上 → ② カテゴリーの拡大 → ③ 販路の拡大 → ④ リピートビジネスの推進
         ↑                                                           │
         └───────────────────────────────────────────────────────────┘
```

**戦略的解釈**:

FY2024-FY2025で家電量販店解禁（販路拡大）とミルクプロテインシリーズ（リピート化）を達成した今、MTGはサイクルの起点である**「ブランド価値の向上」に再投資**している。ReFa GINZAはその象徴的な施策であり、単なる販売拠点ではなく**ブランド体験の発信地**として位置づけられている。

これは「成長の収穫」ではなく、**次の成長サイクルへの種蒔き**と捉えるべきである。

*出典: MTG 2025年9月期 通期決算説明資料（ReFaブランド成長戦略、ReFa GINZA）*

## 5. チャネル戦略分析

### 5.1 段階的チャネル展開

| Phase | 時期 | チャネル | 目的 | 結果 |
|-------|------|---------|------|------|
| 1 | 2019年 | 美容室限定 | プロユースブランディング | ブランド信頼構築 |
| 2 | 2020年 | 百貨店・EC追加 | 高級感維持 + コロナ対応 | EC売上大幅増 |
| 3 | 2019-2023年 | ホテルBtoBtoC | 体験機会創出 | 2,600施設導入 |
| 4 | 2024年 | **家電量販店解禁** | マス拡大 | 売上728億円達成 |

### 5.2 「5年かけた戦略的忍耐」

**従来の美容家電メーカー**:
- 発売直後から家電量販店で販売
- 価格競争に巻き込まれる
- ブランドイメージが「家電」に固定

**ReFaのアプローチ**:
- 5年間は美容室・百貨店限定
- 「美容室で売っているドライヤー」というポジショニング確立
- ブランド価値が定着してからマス展開

**松下社長のコメント**（2019年当時）:
> 「美容室や百貨店で売っているドライヤー」という印象が顧客に定着するまでは、たとえ売れても家電量販店などには絶対に展開しないと決めていた

### 戦略的インサイト

**1. ダイレクト（BtoC）が利益の柱**
- 全利益の約**57%**（106.25億円/186.40億円）を占める
- EC・通販は利益率が高く、成長期に最も貢献

**2. プロフェッショナルは「先行投資→回収」モデル**
- FY2023に利益△11%減: サロンネットワーク拡充に経営資源を投入
- FY2024-25で利益急回復（+51%→+74%）: 投資が売上に転換

**3. リテールストアが急成長**
- FY2023: コロナ後のリアル店舗回帰で利益+98%
- FY2025: 家電量販店本格展開で売上+56%
- 「5年かけた戦略的忍耐」の成果が数字に表れた

## 7. 危機対応分析

### 7.1 2019年の危機

| 項目 | 内容 |
|------|------|
| 問題 | インバウンド依存（売上155億円減少） |
| 原因 | 中国新EC法、韓国不買運動、香港デモ |
| 損失 | 営業利益 -144億円（前年比 -256億円） |
| 状況 | ローラー依存、単一カテゴリ、海外市場依存 |

### 7.2 復活戦略（4つの柱）

#### 柱1: 製品多角化
- ローラー → ヘアケア → バス用品 → コスメ
- 単一製品依存からの脱却
- **結果**: ヘアケア・シャワーが年間100億円超の柱に成長

#### 柱2: 国内市場強化
- インバウンド依存から国内EC・ホテルBtoBtoCへ
- 「体験の場」創出で新規顧客開拓
- **結果**: 国内売上が過去最高を連続更新

#### 柱3: 段階的チャネル解禁
- ブランド価値確立後に家電量販店展開
- 5年かけた戦略的忍耐
- **結果**: 高単価を維持したままマス展開に成功

#### 柱4: 消耗品によるリピート収益化
- シャンプー・トリートメント参入（2024年）
- 機器売りからリピート収益モデルへ
- **結果**: 継続的な収益基盤構築

## 8. まとめ：ReFa復活の成功要因

### 8.1 What（何を推したか）の変遷

| 時期 | 主力製品 | ポジショニング |
|------|---------|---------------|
| 〜2018年 | 美顔ローラー | エステティシャンの手技を再現 |
| 2019年〜 | ドライヤー・アイロン | 美容室の仕上がりを自宅で |
| 2020年〜 | シャワーヘッド | ファインバブルで美肌・美髪 |
| 2024年〜 | ヘアケアコスメ | プロテインで髪に栄養補給 |

### 8.2 How（どう推したか）の変遷

| 時期 | 手法 | 狙い |
|------|------|------|
| 〜2018年 | 口コミ・PR | 認知拡大 |
| 2019年〜 | BtoBtoC（サロン・ホテル） | 体験機会創出 |
| 2022年〜 | インフルエンサー | 共感・自分ごと化 |
| 2024年〜 | マスCM | 購買促進 |

### 8.3 成功の本質

1. **危機を転機に変えた**: インバウンド崩壊を製品多角化のきっかけに
2. **ブランド価値を守った**: 短期売上より長期ブランド価値を優先
3. **体験を重視した**: BtoBtoCで「使ってもらう」機会を創出
4. **段階的に拡大した**: 5年かけてチャネルを慎重に展開

### 8.4 今後の展望：「収穫」ではなく「再ブランディング」

FY2025で売上728億円、営業利益率11%を達成したReFaは、一見「収穫期」に入ったように見える。しかし、MTGの戦略を詳細に分析すると、**次の成長サイクルに向けた「再ブランディング」フェーズ**に入っていることがわかる。

**根拠**:
1. **ReFa GINZA への大型投資**: 2025年11月に銀座に旗艦店をオープン。世界的クリエイターを起用し、"ReFa Amazing Luxury"をコンセプトとしたブランド体験の発信地として位置づけ
2. **研究開発費+53%増**: FY2025の研究開発費は34.23億円で前年比+53%増。収穫期なら投資を抑えるはずが、むしろ積極投資
3. **ブランド成長戦略サイクルの起点回帰**: 「ブランド価値の向上 → カテゴリー拡大 → 販路拡大 → リピート化」のサイクルにおいて、販路拡大・リピート化を達成した今、再び起点の「ブランド価値の向上」に戻っている

**示唆**:
ReFaの戦略は「成長→収穫→衰退」という一方通行ではなく、**循環型の持続成長モデル**を採用している。これは単なる美容家電ブランドを超え、**ラグジュアリービューティーブランドとしてのポジション確立**を目指していることを示している。
