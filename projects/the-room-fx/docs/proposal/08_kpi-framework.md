## 8. KPI目標の算出根拠

### 主要KPIの計算根拠

| 指標 | 目標 | 業界平均 | 算出根拠 |
|------|------|---------|---------|
| **リーチ** | 1,000万 | - | 予算4,000万÷平均CPM¥1,200×1,000≒3,300万imp、FQ3.3でリーチ1,000万 |
| **インプレッション** | 5,000万 | - | YouTube 3,000万 + Meta 1,500万 + PMP 460万 + 検索/Display 40万 |
| **視聴完了率** | 35% | YouTube平均31% | 下記「視聴完了率35%の根拠」参照 |
| **LP流入** | 10万 | CTR 0.3%想定 | 下記「LP流入10万の根拠」参照 |

### 視聴完了率35%の根拠

**業界平均**: Travel & Tourism業界のYouTube視聴完了率は**31%**（Google Ads Benchmark 2025）

**+4ptの内訳**:

| 要因 | 寄与 | 根拠 |
|------|:----:|------|
| プレミアム層ターゲティング | +2pt | 高所得層は広告忍耐度が高い（Nielsen調査） |
| 高品質クリエイティブ | +1pt | ANA支給素材（TVCM転用）は制作品質が高い |
| 関心層への配信 | +1pt | カスタムインテント（business class検索者）は関心が明確 |
| **合計** | **+4pt** | **31% + 4% = 35%** |

**参考実績**: 同業他社（航空会社）のYouTube広告平均視聴完了率は32-34%（社内ベンチマーク）

### LP流入10万の根拠

**CTR想定**: 0.3%（Travel業界ベンチマーク: 0.2-0.4%の中央値）

**計算式**:
```
動画imp 4,500万 × CTR 0.3% = 13.5万クリック
→ 保守的に見積もって10万を目標設定
```

**新製品ローンチ特性**:
- ローンチ初期は話題性でCTR上昇傾向（+0.1pt）
- 一方、認知がない状態からの流入のためスルー率も高い（-0.1pt）
- 相殺して業界平均0.3%を採用
