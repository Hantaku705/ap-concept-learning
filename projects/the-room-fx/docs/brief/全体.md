## 目的（Objective）

1. **ANAブランド認知度向上**（企業名・ブランドイメージの浸透）
2. **ANA国際線の将来的な利用意向醸成**
3. **THE Room FX認知最大化**（PY/Y認知はサブ）

---

## 背景・課題

| 課題 | 詳細 |
|------|------|
| THE Room認知不足 | 777-300ERの「THE Room」はコロナ禍で告知強化できず、認知獲得が不十分 |
| JAL競合 | JALがA350-900/1000をロング路線に積極導入、プロダクト×運賃で付加価値を創出 |
| プロダクト劣後 | 対JAL観点でTC1（北米）・TC2（欧州）エリアでプロダクトが劣後している |

**戦略的意図**: 従来の「Usage（機能・仕様）中心」の訴求から脱却し、**顧客体験メリット（情緒的価値）**を強調したコミュニケーションへ転換

```

```
Global
├── Communication
│   └── 商品認知/イメージ醸成
│       └── プロダクトアップデートを超えた、
│           空の上の空間の付加価値を顧客ベネフィットとして全面展開
│
└── 地域別展開
    ├── TC1 北米（Focus）: ANAブランド認知度向上 + 北米路線プロモーション
    ├── TC2 欧州（Focus）: ANAブランド認知度向上 + 欧州路線プロモーション

## 戦略サマリー

**従来のシートスペック訴求から「情緒的な顧客体験価値」へのシフトが戦略の核心**

- 機能・仕様（Usage）中心 → 顧客体験メリット（Comfort/寛ぎ）中心
- Functionalな「快適」 → 深くくつろぎ、自分をRetreatできるComfort
- シートスペック強調 → 空の上の空間を持つことの付加価値

**競合差別化**: JAL A350に対抗し、THE Room FXでプロダクト×運賃の付加価値を創出
