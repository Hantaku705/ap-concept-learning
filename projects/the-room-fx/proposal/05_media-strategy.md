# メディア戦略 - PMP × ブランドセーフティ × データドリブン

## エグゼクティブサマリー

**アイレップとの差別化**: PMP（プログラマティック保証型）とブランドセーフティを軸に、ANAブランドにふさわしい高品質な配信面を確保します。

| 方針 | 内容 |
|------|------|
| **PMP活用** | プレミアム媒体での配信面確保 |
| **ブランドセーフティ** | MFA回避、ホワイトリスト運用 |
| **トライブベース** | デモグラではなく行動・関心軸でターゲティング |
| **データドリブン** | SNS分析5,795件に基づく媒体選定 |

### アイレップとの戦略比較

| 観点 | アイレップ（予想） | AnyMind（提案） |
|------|------------------|-----------------|
| **ターゲティング** | デモグラフィック中心（年齢・性別・地域） | トライブベース（行動・関心・価値観） |
| **媒体選定** | Google/Meta標準枠中心 | PMP活用でプレミアム媒体を確保 |
| **配信面品質** | オープンオークション依存 | ホワイトリスト＋MFA回避 |
| **データ活用** | 媒体データのみ | SNS分析5,795件＋GA4連携 |
| **最適化** | 媒体別サイロ運用 | クロスプラットフォーム統合運用 |

**AnyMindの優位性**:
- **トライブターゲティング**: 「35-54歳男性」ではなく「ビジネス×テック関心層」で精度向上
- **PMP × ブランドセーフティ**: ANAブランドにふさわしい配信面を保証
- **データドリブン**: 5,795件のSNS投稿分析に基づく媒体・メッセージ設計

## 1. メディアミックス概要

### 予算配分（4,000万円）

| カテゴリ | 配分 | 金額 | 役割 |
|---------|:----:|-----:|------|
| **YouTube** | 40% | 1,600万円 | 動画認知、ブランド訴求 |
| **Meta（FB/IG）** | 30% | 1,200万円 | リーチ、エンゲージメント |
| **Google検索/Display** | 15% | 600万円 | 検索インテント、リマーケティング |
| **PMP / プレミアム** | 15% | 600万円 | ブランドセーフティ、高品質面 |

### 媒体選定理由（SNS分析5,795件に基づく）

| トライブ | 最適媒体 | 選定根拠 |
|---------|---------|---------|
| **Business Traveler** | YouTube、LinkedIn | エンゲージメント最高（91,474）、ビジネス層は動画レビューで情報収集。投稿数616件。 |
| **Tech/Digital** | YouTube、Instagram | 最大ボリューム（922件）、テック系はYouTubeレビュー・IG Reelsを重視。 |
| **Travel Enthusiast** | YouTube、航空専門メディア | 唯一ポジティブセンチメント（+0.54）、詳細スペックを求める傾向。 |

**なぜMeta 30%か？**
- Z/Y世代（ブリーフ指定ターゲット）の主要接触媒体
- Instagram ReelsはTech/Digital層で高エンゲージメント
- Facebook In-FeedはBC渡航層・訪日検討層に効果的

**なぜYouTube 40%か？**
- 全トライブで共通して高い接触率
- 30秒動画でブランドストーリーを伝達可能
- 視聴完了率が他媒体より高い（業界平均31%）

## 2.6 追加媒体の検討

Google/Meta以外のプレミアム媒体について検討した結果を以下に記載。

| 媒体 | 検討結果 | 理由 |
|------|:-------:|------|
| **Netflix広告** | 見送り | CPM $30-50と高額。3月単月では投資効率悪い。年間プランで再検討推奨 |
| **Spotify** | 見送り | 視覚訴求が必要なプロダクト。オーディオのみでは「THE Room FX」の魅力を伝達困難 |
| **Reddit** | 見送り | r/aviation等の航空コミュニティはブランドセーフティリスクあり。荒れやすい傾向 |
| **TikTok** | 限定採用 | Z世代接触に有効。ティーザー期間のみ50万円を配分（Phase1） |
| **The Points Guy** | **採用（PMP枠）** | マイラー・Travel Enthusiastに直接リーチ。航空レビューの信頼性高 |
| **One Mile at a Time** | **採用（PMP枠）** | 航空レビュー専門。ビジネスクラスの詳細レビューで高エンゲージメント |

**年間プラン（4月以降）での検討候補**:
- Netflix広告（長期ブランディング）
- Connected TV（Hulu、Disney+等）
- ポッドキャスト広告（ビジネス系番組）

## 3. ブランドセーフティ対策

### 3.1 MFA（Made For Advertising）回避

| 対策 | 内容 |
|------|------|
| **ホワイトリスト運用** | 事前承認した配信面のみに配信 |
| **ブロックリスト設定** | MFAサイト、低品質サイトを除外 |
| **IAS/DV連携** | 第三者検証ツールでブランドセーフティを担保 |

### 3.2 除外カテゴリ

| カテゴリ | 理由 |
|---------|------|
| 政治・宗教 | ブランド毀損リスク |
| 事故・災害 | 航空会社として不適切 |
| 成人向けコンテンツ | ブランドイメージ保護 |
| フェイクニュース | 信頼性の問題 |

### 3.3 ビューアビリティ基準

| 指標 | 基準 |
|------|------|
| 動画ビューアビリティ | 70%以上 |
| ディスプレイビューアビリティ | 60%以上 |
| 完全視聴可能率 | 50%以上 |

## 6. レポーティング体制

### 動画系指標（ブリーフ要件準拠）

| 指標 | 定義 | 目標 |
|------|------|------|
| CPM | 1,000インプレッションあたり単価 | $8以下（北米） |
| 表示回数 | 広告表示回数 | 5,000万以上 |
| 視聴完了率 | 視聴完了数÷視聴開始数 | 35%以上 |
| 視聴完了数 | 動画を最後まで視聴した回数 | 1,000万以上 |
| 視聴完了単価 | 予算÷視聴完了数 | $0.03以下 |
| クリック数 | LP誘導クリック | 10万以上 |
| リーチ数 | ユニークリーチ | 1,000万以上 |
| FQ数 | 平均接触回数 | 3-5回 |

**視聴定義**:
- YouTube: 30秒以上または最後まで視聴
- Meta: 3秒以上（ThruPlay: 15秒以上）
- LinkedIn: 2秒以上

### バナー系指標

| 指標 | 定義 | 目標 |
|------|------|------|
| CPM | 1,000インプレッションあたり単価 | $6以下 |
| 表示回数 | 広告表示回数 | 2,000万以上 |
| クリック数 | LP誘導クリック | 5万以上 |
| クリック率 | クリック数÷表示回数 | 0.25%以上 |
| リーチ数 | ユニークリーチ | 500万以上 |
| FQ数 | 平均接触回数 | 3-5回 |

### レポート提出スケジュール

| レポート | 頻度 | 内容 |
|---------|------|------|
| 速報レポート | 週次 | KPI進捗、異常値アラート |
| 詳細レポート | 月次 | 媒体別分析、改善提案 |
| 最終レポート | キャンペーン終了後 | 総括、次回への提言 |

### 対応方針

| 項目 | 方針 |

**注記**: ローカライズ制作費は本予算（4,000万円）外。ANA様にて手配、または別途見積もりにて対応。
