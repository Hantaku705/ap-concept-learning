# 「何ゲーか」分析 - 市場構造の本質を特定するフレームワーク

市場構造（ゲーム構造）の本質を特定し、勝者を分析して再現手順を導出する。

---

## 本質とは

| 用語 | 定義 |
|------|------|
| **本質** | 市場構造（ゲーム構造）における勝利条件 |
| **何ゲーか** | その市場で勝つために必要な要因を一言で表現 |
| **勝者分析** | 勝ってるプレイヤーから勝因を抽出し、再現手順を導出 |

### 核心原則

> **本質を見つけるには、勝ってる人を見つけることが重要。**
>
> 勝者が存在する = ゲーム構造が存在する = 勝利条件が存在する

---

## フレームワーク（4 Phases）

```
Phase 1: ゲーム特定    → 「〇〇ゲー」命名
    ↓
Phase 2: 勝者発見      → Top3〜5リスト
    ↓
Phase 3: 勝因分析      → 表層→構造→本質の3層分析
    ↓
Phase 4: 再現手順      → 具体的アクションプラン
```

---

## Phase 1: ゲーム特定

### 質問リスト

| 質問 | 目的 |
|------|------|
| 誰が、何を、誰に提供しているか？ | 市場の定義 |
| 何をもって「勝ち」とするか？ | 勝利条件の定義 |
| 勝利条件を一言で言うと？ | 「〇〇ゲー」命名 |

### 命名テンプレート

```
[勝利条件の本質] + ゲー

例:
- 移動平均ゲー（前の動画が次に影響）
- 初動ハックゲー（初速が全てを決める）
- ループゲー（A→B→C→A の好循環）
- PKGゲー（セット売りでマージン確保）
- 箱ゲー（場を作る者が勝つ）
- ネットワークゲー（人脈・リソースの量が勝負）
```

---

## Phase 2: 勝者発見

### 勝者の定義

| 条件 | 説明 |
|------|------|
| **定量的に優位** | 売上、再生数、フォロワー、シェア等で上位 |
| **持続的に勝利** | 一発屋ではなく継続的に成果を出している |
| **再現性がある** | 偶然ではなく、構造的に勝っている |

### 勝者発見の方法

1. **業界ランキング・メディア**: 売上Top10、フォロワーランキング等
2. **競合分析ツール**: SimilarWeb、Meltwater、各SNS分析ツール
3. **業界関係者からのヒアリング**: 「あそこが強い」という声
4. **成功事例の逆引き**: 「〇〇で成功した」→ その人を調査

### 出力テンプレート

| 勝者 | 数字 | 強み |
|------|------|------|
| 会社/人名 | 売上〇億円/月、FW〇万人 | 一言で強みを表現 |

---

## Phase 3: 勝因分析（3層モデル）

```
表層的勝因（What）: やっていること（施策・戦術）
    ↓ なぜそれが効くのか？
構造的勝因（Why）: メカニズム・因果関係
    ↓ なぜ他は真似できないのか？
本質的勝因（How）: 資源・ケイパビリティ・参入障壁
```

### 各層の質問

| 層 | 質問 | 例 |
|----|------|-----|
| 表層 | 何をやっているか？ | 毎日投稿、コメント返信、広告配信 |
| 構造 | なぜそれが効くのか？ | アルゴリズムが〇〇を優遇するから |
| 本質 | なぜ他は真似できないか？ | 40名の編集者体制、独自ネットワーク |

### 出力テンプレート

| 層 | 勝因 | 詳細 |
|----|------|------|
| 表層（What） | 施策名 | 具体的な行動 |
| 構造（Why） | メカニズム | なぜ効くのか |
| 本質（How） | 参入障壁 | 真似できない理由 |

---

## Phase 4: 再現手順

### 再現可能性の判定

| 勝因 | 真似可能？ | 必要リソース |
|------|-----------|-------------|
| 表層的勝因 | ◎ 可能 | 知識・実行力 |
| 構造的勝因 | ○ 条件付き | 時間・資金 |
| 本質的勝因 | △ 困難 | 大規模投資・時間 |

### 再現手順の作成

1. **真似できる要素を抽出**: 表層的勝因から
2. **必要リソースを特定**: 人・金・時間・スキル
3. **優先順位を決定**: 効果 × 実行難易度
4. **アクションプランを作成**: 具体的な実行ステップ

### 出力テンプレート

| # | アクション | 必要リソース | 期待効果 | 優先度 |
|---|-----------|-------------|---------|--------|
| 1 | 具体的行動 | 人・金・時間 | 期待成果 | 高/中/低 |

## チェックリスト

### ゲーム特定時
- [ ] 市場の定義が明確か（誰が、何を、誰に）
- [ ] 勝利条件が定量的か（数字で測れるか）
- [ ] 「〇〇ゲー」として一言化できたか

### 勝者発見時
- [ ] Top3〜5の勝者をリストアップしたか
- [ ] 各勝者の数字（売上、FW等）を把握したか
- [ ] 一発屋ではなく持続的な勝者か確認したか

### 勝因分析時
- [ ] 表層的勝因（やっていること）を列挙したか
- [ ] 構造的勝因（なぜ効くか）を説明できるか
- [ ] 本質的勝因（真似できない理由）を特定したか

### 再現手順作成時
- [ ] 真似できる要素を抽出したか
- [ ] 必要リソース（人・金・時間）を見積もったか
- [ ] 優先順位をつけたか
- [ ] 具体的なアクションプランを作成したか
