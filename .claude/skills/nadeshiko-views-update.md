# NADESHIKO再生数更新スキル

NADESHIKOの再生数シートが更新された際、ファイル構造の解析をスキップして即座にサマリーを生成する。

---

## ファイル構造（固定・毎回の解析不要）

| 行 | 内容 |
|----|------|
| 2 | 全体（目標/現状/進捗率） |
| 3-5 | TikTok（目標/現状/進捗率） |
| 6-8 | IG（目標/現状/進捗率） |
| 9-11 | YT（目標/現状/進捗率） |
| 12-14 | Total（目標/現状/進捗率） |
| 15 | 投稿数 |
| 17以降 | 投稿データ |

## 実行手順

1. ユーザーがCSVパスを提供
2. `Read(limit: 16)` でサマリー行のみ取得（構造解析スキップ）
3. 行2から全体進捗を抽出
4. 行12-14からTotal（アカウント別）を抽出
5. 以下のフォーマットでレポート生成
